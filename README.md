# realtime-delivery-microservices
Système de livraison en temps réel basé sur une architecture microservices avec Node.js

## Description
Ce projet est un mini-projet académique basé sur une architecture microservices en Node.js.

L'application représente un système de livraison en temps réel. Elle permet de gérer les commandes, les livreurs et les livraisons à travers plusieurs microservices indépendants.

## Technologies utilisées
- Node.js
- Express.js
- gRPC
- GraphQL
- Kafka
- SQLite3
- RxDB
